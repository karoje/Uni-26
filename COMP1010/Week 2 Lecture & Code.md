 Indexing always starts from 0, e.g. Row 0 and Column 0
`array [i] [k]` gives the number in `row [i]` and `column [k]`

`int[] b = arr;` --> b is a reference copy of the array "arr"

`int + string = string`

Loops consist of an expression which determines whether or not the code within the loop should be repeated, along with a block containing the code to be repeated. e.g.
```java
while (_expression_) {
	//content of the block wrapped in curly brackets
}
```
